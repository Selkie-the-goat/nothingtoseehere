## Supported Formats

- JPG/JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Image Recommendations

- **Size**: Square images work best (e.g., 500x500px or 1000x1000px)
- **Format**: JPG is recommended for smaller file sizes
- **Quality**: High quality images will look better on the vinyl buttons
